# Quick Start

1. Make sure Git & NodeJS are installed
2. `git clone https://github.com/syahmifauzi/react-projects.git`
3. `cd react-projects/<project-name>/`
4. `npm install`
5. `npm start`
6. Open web browser & navigate to `http://localhost:3000`
