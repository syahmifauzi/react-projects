## Next.js Starter Kit

### This Starter Kit Includes:

- Next.js
- TypeScript
- TailwindCSS
- Prettier
- Absolute Import
- SEO & Layout Components
- Custom 404 Page
- React Icons

### Quick Start

```bash
$ npx create-next-app -e https://github.com/syahmifauzi/nextjs-starter-kit my-app
$ cd my-app
$ npm run dev

# open http://localhost:3000
```
